Project Title

Development of Blockchain-Based Security Sharing Scheme for Research and Medical Data with Fine-Grained Access Control

Project Summary

This project proposes and implements a blockchain-based security framework for the secure sharing of medical and research data among hospitals, doctors, and researchers. By leveraging smart contracts, the system enforces fine-grained access control policies that ensure sensitive patient data is accessed strictly by authorized entities. The solution addresses key challenges such as data privacy, trust, and unauthorized access in healthcare information systems. Performance evaluation results demonstrate that the proposed approach is efficient, secure, and scalable, making it suitable for real-world deployment in collaborative medical and research environments.
